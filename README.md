# Iron Frame PWA

Installeerbare trainingsapp voor Athletic Kettlebell Hypertrophy.

## Publiceren met GitHub Pages

1. Maak een lege GitHub-repository.
2. Upload alle bestanden uit deze map rechtstreeks naar de root van de repository.
3. Open op GitHub `Settings > Pages`.
4. Kies `Deploy from a branch`, branch `main` en map `/ (root)`.
5. Open de gepubliceerde HTTPS-link op Android en kies `Installeren` of `Toevoegen aan startscherm`.

Alle voortgang wordt lokaal in de browser of geïnstalleerde app opgeslagen.
